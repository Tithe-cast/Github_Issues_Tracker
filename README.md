1.What is the difference between var, let, and const? 
In JavaScript, var, let, and const are used to declare variables. var is the older method and has function scope, meaning it can be accessed anywhere inside the function and it can be redeclared and reassigned. let is block-scoped, so it works only inside the block where it is declared, and it can be reassigned but not redeclared. const is also block-scoped like let, but its value cannot be changed after it is assigned, so it is used for variables that should remain constant.
2.What is the spread operator (...)?
The spread operator (...) is used to expand elements of an array or properties of an object into individual elements. It helps to copy arrays or objects, merge multiple arrays, and pass multiple values easily. This makes the code shorter and easier to read.
3.What is the difference between map(), filter(), and forEach()? 
The map(), filter(), and forEach() methods are used to work with arrays. map() creates a new array by applying a function to every element. filter() creates a new array with only the elements that satisfy a given condition. forEach() simply runs a function for each element in the array but does not return a new array.
4.What is an arrow function? 
An arrow function is a modern and shorter way to write functions in JavaScript using the => syntax. It makes the code more concise and is commonly used in array methods and modern JavaScript programming.
5.  What are template literals?
Template literals are used to create strings in JavaScript using backticks ( ` ). They allow developers to easily insert variables or expressions inside a string using ${ } and also support writing multi-line strings, which improves code readability.
